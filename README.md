# Retail Sales Analysis using SQL

## Problem Statement
Analyze retail sales data to identify revenue trends, top-performing products, and customer behavior.

## Tools Used
- SQL

## Steps Performed
- Data cleaning and preprocessing
- Data aggregation using GROUP BY
- Joins to combine multiple tables
- Trend analysis

## Key Insights
- Top-selling products contribute major revenue
- Certain months show higher sales trends
- Some regions underperform compared to others

## Conclusion
This analysis helps in understanding sales performance and supports better business decisions.
